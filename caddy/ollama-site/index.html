Random Studios

Ollama Proxy

Use https://ollama.random-studios.net/v1 as your OpenAI-compatible base URL.

OpenCode

In OpenCode, type /connect and select ollama.random-studios.net.

qwen3.5:latest is the recommended shared default when multiple people are using the box at the same time.

Auth

Requests to the API require a bearer token.

Authorization: Bearer <your-token>

The homepage is public. API endpoints stay protected.

Qwen 3.5 Notes

  • qwen3.5:latest currently points to the 9B variant in the Ollama library.
  • The 0.8B, 2B, 4B, 9B, and 27B Qwen 3.5 variants are dense models.
  • The larger 35B-A3B and 122B-A16B variants are MoE models, where only part of the model is active per token.
  • Approximate Ollama footprint: qwen3.5:latest is listed at 11 GB; qwen3.5:27b is listed at 30 GB.
